# **App Name**: SmartInvest AI

## Core Features:

- User Authentication: User authentication with 'Add User' and 'Select User' options, including secure storage of user data.
- Real-time Stock Data: Real-time stock data display, sourced from external APIs (including https://twelvedata.com/), for both Indian (NSE/BSE) and US stock markets, with a ticker displaying world market overview and stock exchange data, as well as multiple Demat and Bank account integration
- Custom Screeners: Customizable screeners allowing users to define criteria for identifying investment opportunities.
- AI-Powered Stock Analysis: AI-powered analysis tool that helps investors with automated fundamental and technical analysis, including SWOT analysis, triggered by selecting one of multiple prompts, where prompts instruct a LLM to tool over company data fetched from the internet
- Uptrend Stock Screening: Integration with an 'Uptrend Finder' to filter and categorize stocks based on growth potential, with color-coded visual representation.
- Smart Watchlist with Alerts: Smart Watchlist to save preferred stocks with price and PE data and options to delete stocks and set custom alerts via email or Telegram based on predefined time periods and threshold types.
- Core Portfolio Management: Portfolio management with stocks and fixed deposits add, edit and remove entry, which also supports multiple bank accounts and multiple demat account. Will automatically give reminder if they're reaching towards the expiry.

## Style Guidelines:

- Primary color: HSL 210, 75%, 50% - A vibrant blue (#2693E8) reflecting trust and financial acumen.
- Background color: HSL 210, 20%, 95% - Very light blue (#F0F6FA) for a clean, spacious feel that focuses user attention.
- Accent color: HSL 180, 60%, 40% - A contrasting teal (#29AFAF) for highlighting important calls to action.
- Headline font: 'Space Grotesk' (sans-serif) for headlines and short amounts of body text, lends a tech-forward, modern aesthetic.
- Body font: 'Inter' (sans-serif) as body font in conjunction with 'Space Grotesk' in order to ensure readability in longer texts.
- Use financial and analytical iconography for a professional look.
- A clean, structured layout with clear sections for research, portfolio, and watchlist.
